# AI-Agent-Driven Swarm Navigation System

This project implements a multi-agent autonomous quadrotor swarm system. It leverages the concepts of the EGO-Planner framework, utilizing ESDF-free Gradient-based Optimization and B-spline trajectories for high-performance navigation.

## Core Features
- **AI-Agent Orchestration**: Distributed intelligence model for adaptive decision-making.
- **ESDF-Free Collision Avoidance**: Ultra-low latency gradient-based optimization pipeline.
- **B-Spline Trajectories**: Smooth, kinodynamically feasible path generation.
- **Scalable Architecture**: Modular design supporting multi-drone coordination and SITL integration.

## Project Structure
- `swarm_system.md`: Preliminary Design Review (PDR) and technical documentation.
- `.agent/`: Agent skills and configuration for the AI assistant.

## Quick Start
1. **PDR Review**: Read `swarm_system.md` to understand the mathematical and architectural foundations.
2. **Development**: Follow the guidelines in the PDR for C++, ROS 2, and MAVLink implementation.
